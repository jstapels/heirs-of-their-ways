# Temporal Dust
*Wondrous item, uncommon (consumable)*

Shimmering amber dust collected from flickerswarm wings. Three vials are found in the hall.

**Uses:**
- As a material component for a Divination spell, this dust can replace components worth 100 gp or less.
- As an action, throw a vial to create a 10-foot radius cloud that lasts 1 round. Creatures in the cloud must succeed on a [[/save con dc 12]]{DC 12 Constitution save} or be outlined in amber light (as *Faerie Fire*) for 1 minute.
